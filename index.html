DRILLS




Fork me on GitHub

This program creates and speaks workout drills.

It will construct the drill randomly from the listed activities, read the activities out-loud until the round length is up, then start again after the break.

There are some pre-defined activity categories, but you can edit these and create your own routine that the program will read.

You can also modify the activity interval lengths, the round length, and the break length. Also, don't forget to hydrate.

Sorry, your browser doesn't support text-to-speech.
